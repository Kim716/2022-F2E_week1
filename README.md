## Introdution
2022 The F2E 4th 第一關「視差滾動互動式網站」的作品

這是我第一次進行比較複雜的網頁設計，過程中練習了架構流程、動畫技術，在 RWD 上仍有不足，目前覺得桌機大視窗版的畫面觀賞體驗最佳，平板、手機視窗的仍可以再優化。謝謝設計師可愛有趣的設計稿！
